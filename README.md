# Futura Serviços Mecânicos

Site institucional estático para a Futura Serviços Mecânicos, com foco em manutenção de máquinas pesadas, serviços eletrônicos e diagnóstico computadorizado.

## Estrutura

```text
.
├── index.html
├── assets/
│   ├── css/
│   ├── js/
│   └── images/
│       ├── brand/
│       ├── hero/
│       ├── services/
│       └── originals/
└── tools/
```

## Como abrir

Abra `index.html` diretamente no navegador.

Para testar no celular pela mesma rede Wi-Fi, execute:

```bat
iniciar-site-celular.bat
```

Depois acesse no celular:

```text
http://192.168.2.34:4174/
```
